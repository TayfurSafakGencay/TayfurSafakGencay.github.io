Subway Surfers (Prototype)

A case project requested by a company to be completed within a week. They asked for a Subway Surfers clone with limited assets. I developed the project using the StrangeIoC framework.

Technologies and Features Expertise:

 •  StrangeIoC
 •  Highest Score System
 •  Enhanced garbage collector
 •  Object Pooling
 •  An infinite, continuously running game
 •  Randomly generated game map
